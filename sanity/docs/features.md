## Sanity Studio Features (No Shopify)

This project uses Sanity as a headless CMS for website content only.
There is no Shopify integration in this template.

### Included areas

- `/schemas/documents/`: top-level content documents.
- `/schemas/singletons/`: singleton documents such as site settings.
- `/schemas/objects/`: reusable object and module schemas.
- `/queries/`: GROQ queries used by Next.js.
- `/desk/`: custom desk structure.

### Notes

- Keep schemas focused on editorial content for the site.
- Remove unused schema types before launching a new project.
- Keep query fragments reusable and colocated by feature.
